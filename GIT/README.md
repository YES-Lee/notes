## Git 常用指令

### 基本操作
#### 版本管理
1. 初始化一个版本库
    ```bash
    # 进入项目目录下
    git init
    ```
2. 将文件添加到版本库
    ```bash
    # 添加单个/多个文件
    git add filename1 filename2
    # 添加目录下所有文件
    git add .
    ```
3. 提交一个版本记录
    ```bash
    git commit -m "commit message" # 每次提交都需要写上提交说明
    ```
4. 查看版本/提交记录
    ```bash
    git log
    ```
